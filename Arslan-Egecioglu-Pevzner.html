The dynamic programming approach by Abdullah N. Arslan, Ömer Eğecioğlu and Pavel A. Pevzner (2001) computes the best normalized local alignments of two sequences. It identifies the local alignment that is best preserved with respect to the length of the aligned subsequences.

To find a normalized local alignment, the fractional programming algorithm by Werner Dinkelbach (1967) is used in concert with the 'standard' local alignment algorithm by Smith-Waterman (1981). A fractional programming is a machine learning approach to solve problems in which the objective function $ns(\mathcal{A})$ is a fraction.

The normalized local alignment algorithm tries to find the alignment $\mathcal{A}$ of two sequences with the highest normalized score $$\quad\lambda^* = ns(\mathcal{A}) = \frac{s(\mathcal{A})}{⟦\mathcal{A}⟧ + L}$$ where $⟦\mathcal{A}⟧$ is the length of the aligned subsequences (excluding gaps) and $L$ is a parameter to control the length-dependence of the normalized similarity score.

Given a similarity scoring function $s(a_i,b_j)$ for two sequences $a$ and $b$. To find $\lambda^*$ and the according best normalized alignment $\mathcal{A}$, the Smith-Waterman algorithm is executed iteratively using an iteration-specific scoring-function $s^r$ (see below). In iteration $r>0$, $s^r$ depends on the $\lambda^{r-1}$ estimate of the previous iteration, where $\lambda^{0} = 0$. Given this, the best local alignment $\mathcal{A}^r$ with score $S^r$ using $s^r$ is identified and the new lambda estimate $\lambda^r$ is computed by $\lambda^r= S^r/(⟦\mathcal{A}⟧+L)$. This is iterated until $\lambda$ converges or a certain number of iterations is exceeded.

For a given input, the according tables and scoring functions are shown for up to 5 iterations (or convergence) along with a list of according optimal local alignments. On table cell selection, optimal traceback options are shown. On alignment selection, an according traceback path is visualized in the table.
